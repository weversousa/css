```html
/* arquivo.html */

<button>Enviar</button>
```

Ao passar o cursor por cima do elemento

```css
/* arquivo.css */

button:hover {
    background-color: green;
}
```



***

Ao clicar no elemento

```css
/* arquivo.css */

button:active {}
```

***

Ao deixar o cursor emcima do elemento

```css
/* arquivo.css */

input:focus {}
```
